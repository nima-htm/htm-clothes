# نرم‌افزار حسابداری فروشگاه پوشاک

یک نرم‌افزار حسابداری دسکتاپ برای مدیریت فروشگاه‌های پوشاک

## ویژگی‌ها

- **مدیریت کالا**: تعریف، ویرایش و دسته‌بندی کالاها (پارچه، شلوار)
- **مدیریت طرف حساب‌ها**: ثبت مشتریان و فروشندگان
- **فاکتور فروش**: ثبت فاکتور فروش با قابلیت جستجوی کالا و مشتری
- **فاکتور خرید**: ثبت فاکتور خرید و افزایش موجودی انبار
- **گزارش‌گیری**: گزارش‌های متنوع از فروش، خرید، موجودی و عملکرد اشخاص
- **خروجی اکسل**: امکان خروجی گرفتن از گزارش‌ها

## معماری نرم‌افزار

```
PySide6 (UI)
    ↓
SQLAlchemy (ORM)
    ↓
PostgreSQL/SQLite (Database)
```

## نصب و راه‌اندازی

### نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### اجرای برنامه

```bash
cd htm_clothes
python main.py
```

## تغییر دیتابیس

### استفاده از SQLite (پیش‌فرض)

```python
db = DatabaseManager("sqlite:///nima_clothes.db")
```

### استفاده از PostgreSQL

```python
db = DatabaseManager("postgresql://user:password@localhost/nima_clothes")
```

### استفاده از MySQL

```python
db = DatabaseManager("mysql+pymysql://user:password@localhost/nima_clothes")
```

## ماژول‌ها

### 1. مدیریت کالا (Product Module)
- تعریف کالا با کد یکتا
- دسته‌بندی: پارچه، شلوار
- زیردسته: جین، رسمی، کتان، ...
- قیمت خرید و فروش
- محاسبه خودکار موجودی از روی گردش کالا

### 2. مدیریت طرف حساب‌ها (Party Module)
- ثبت مشتری و فروشنده
- نوع: مشتری، فروشنده، هر دو
- جستجو با autocomplete در فاکتورها

### 3. فاکتور فروش (Sales Invoice)
- انتخاب مشتری
- افزودن کالا با کد یا جستجو
- محاسبه خودکار قیمت
- کاهش موجودی انبار
- چاپ فاکتور

### 4. فاکتور خرید (Purchase Invoice)
- انتخاب فروشنده
- افزودن کالا
- افزایش موجودی انبار

### 5. گزارش‌گیری (Reports)
- گزارش موجودی کالا
- گزارش فروش بر اساس تاریخ
- گزارش خرید بر اساس تاریخ
- گزارش عملکرد اشخاص
- خروجی CSV

## جریان کاری

### فروش
1. انتخاب مشتری
2. افزودن کالاها (با کد یا سرچ)
3. محاسبه خودکار جمع و تخفیف
4. ذخیره در دیتابیس
5. کم شدن موجودی
6. امکان چاپ

### خرید
1. انتخاب فروشنده
2. افزودن کالا
3. افزایش موجودی
4. ثبت فاکتور

### گزارش
1. انتخاب نوع گزارش
2. اعمال فیلتر (تاریخ، نام، کالا)
3. نمایش نتایج
4. خروجی اکسل/چاپ
